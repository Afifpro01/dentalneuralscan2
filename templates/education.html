{% if session.get('language') == 'id' %} Tips Perawatan Gigi Sehari-hari {% else %} Daily Dental Care Tips {% endif %}

{% if session.get('language') == 'id' %} Sikat Gigi dengan Benar {% else %} Brush Properly {% endif %}

{% if session.get('language') == 'id' %} Sikat gigi Anda dua kali sehari selama dua menit menggunakan pasta gigi berfluoride. Gunakan sikat dengan bulu halus dan dengan teknik melingkar lembut. {% else %} Brush your teeth twice daily for two minutes using fluoride toothpaste. Use a soft-bristled brush and a gentle circular technique. {% endif %}

{% if session.get('language') == 'id' %} Gunakan Benang Gigi {% else %} Floss Daily {% endif %}

{% if session.get('language') == 'id' %} Bersihkan sela-sela gigi setiap hari dengan benang gigi untuk menghilangkan plak dan sisa makanan yang tidak bisa dijangkau sikat gigi. {% else %} Clean between your teeth daily with dental floss to remove plaque and food debris that your toothbrush can't reach. {% endif %}

{% if session.get('language') == 'id' %} Perhatikan Pola Makan {% else %} Watch Your Diet {% endif %}

{% if session.get('language') == 'id' %} Batasi makanan dan minuman manis. Hindari ngemil di antara waktu makan dan minum banyak air putih. {% else %} Limit sugary foods and drinks. Avoid snacking between meals and drink plenty of water. {% endif %}

{% if session.get('language') == 'id' %} Pemeriksaan Rutin {% else %} Regular Check-ups {% endif %}

{% if session.get('language') == 'id' %} Kunjungi dokter gigi Anda setiap enam bulan untuk pemeriksaan dan pembersihan profesional. {% else %} Visit your dentist every six months for check-ups and professional cleaning. {% endif %}
